Необходимо реализовать текстовый редактор на C# или Visual Basic .NET с перечисленными ниже возможностями:

(*) открытие/сохранение/переоткрытие файлов;
(*) ввод текста (поддержка delete, insert, backspace);
(*) навигация по тексту (стрелки, home, end, page up/down);
(*) подсветка синтаксиса кода на Java (ключевые слова, идентификаторы, комментарии, подсветка парных скобок — при помещении каретки на одну из скобок, в редакторе должна подсвечиваться парная);
(*) поддержка выделения мышью и клавиатурой;
(*) поддержка работы с буфером обмена;
(*) поддержа больших файлов (редактор должен поддерживать работу с файлами размера 200Кбайт без заметного замедления работы; возможно придется делать инкрементальную подсветку синтаксиса);

При реализации запрещается пользоваться стандартными компонентами редактора. Редакторная компонента должна быть реализована самостоятельно.
